Presentacion Personal

Soy un estudiante de 8vo semestre de Ingeniería de Sistemas en la Universidad Francisco de Paula Santander, tengo conocimientos en las áreas de programación web, gestión de base de datos y sistemas operativos. orientado al desarrollo y gestión de proyectos informáticos. Poseo facilidad para el aprendizaje de nuevas tecnologías y gran adaptabilidad a nuevos entornos de trabajo.

Hasta la actualidad he participado en varios proyectos informáticos universitarios como aplicaciones web y aplicaciones basadas en microservicios enfocados en las siguientes etapas análisis, diseño, desarrollo, riesgos, calidad y testing.

Para desarrollar estos proyectos uso como base al entorno de Node.js en conjunto con Express que me ayuda a gestionar las peticiones a la base de datos para tener un mejor manejo en el Backend También tengo un amplio conocimiento en el lenguaje CSS que acompañado de los lenguajes HTML y JavaScript puedo desarrollar aplicaciones web responsivas y dinámicas para que tengan un Frontend más atractivo.

En mi ultimo proyecto implementé el framework React en el cual ya tengo buenas bases y pienso seguir aprendiendo para volverlo mi framework principal para el Frontend de mis futuros proyectos.

Mi objetivo es mantenerme en constante aprendizaje conociendo nuevas tecnologías, haciendo cursos y a su vez obtener experiencia para cuando obtenga mi título poder ejercer como ingeniero de sistemas en un cargo al nivel de mis habilidades.
